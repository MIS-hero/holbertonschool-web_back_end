# Pagination

 the process of dividing large datasets or content into smaller, manageable chunks or separate pages, commonly used in web design (e.g., search results, product listings) to improve performance, usability, and loading times. It facilitates navigation through "next/previous" buttons or numbered links and is also defined as the sequence of page numbers in books.

## Types

- Offset Pagination
- Keyset Pagination
- Seek Pagination
